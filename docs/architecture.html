计算图

  • Tensor 节点，最多 5 条源边
  • 拓扑排序决定执行顺序
  • 按 layer_id 分组管理
  • 支持 DOT 格式导出可视化
  • 每次前向传播动态解析维度

执行器

  • prefill(tokens) — 批量预填充，seq_len = N
  • decode_step(token) — 单步解码，seq_len = 1
  • append_tokens(tokens) — 多轮对话续写
  • 遍历计算图，按 dtype / device 分发算子
  • 自动插入跨设备内存拷贝

分页 KV 缓存

  • 每层一个 PagedAttentionManager
  • BlockPool 空闲链表，固定大小块
  • 每页一个 PageEntry {k_block_id, v_block_id}
  • append_kv_pages() 将 K/V 散射写入缓存池
  • 因果掩码：kv_pos ≤ q_start + q_pos

注意力机制

  • PagedAttention — 分页 KV 缓存 + 在线 Softmax
  • FlashAttention — 连续 Q/K/V + 在线 Softmax
  • 支持 GQA：num_kv_groups = n_heads / n_kv_heads
  • 当前约束 head_dim == 128（着色器优化）
  • Warp / Subgroup 归约计算点积

Vulkan 管线

  • VulkanContext 单例 — 设备 + 队列 + 命令池
  • 启动时预先创建所有 Pipeline
  • registerOp() — 着色器模块 + 管线缓存
  • 录制 → 提交 → 等待 → 清理 生命周期
  • 延迟释放描述符集以支持批处理

构建系统

  • python build.py --shader — GLSL → SPIR-V → C++ 头文件
  • python build.py --vulkan — Vulkan 全量构建
  • CMake + Ninja / Unix Makefiles
  • C++23，GCC 15+ / Clang 20+
  • 编译期后端开关：BACKEND_CPU / CUDA / VULKAN / SYCL
